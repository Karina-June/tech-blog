## Installations
To install MySQL 2
```bash
npm install --save mysql2
```

To install Sequelize 
```bash
npm i sequelize
```

To install dotenv
```bash
npm i dotenv --save
```

To install Express Handlebars
```bash
npm i express-handlebars
```
To install bcrypt
```bash
npm i bcrypt
```

Application will be invoked using:
```bash
npm run start
````
